# Brand Logos

This folder contains the brand logo images for the "Shop by Brand" section.

## Required Logo Files

Please save the following logo files in this directory:

1. **apple.png** - Apple/iPhone logo
2. **samsung.png** - Samsung logo
3. **oneplus.png** - OnePlus logo
4. **vivo.png** - Vivo logo
5. **oppo.png** - Oppo logo

## Logo Specifications

- **Format**: PNG (with transparent background recommended)
- **Size**: Recommended 256x256 pixels or larger (will be scaled to 64x64px in the UI)
- **Background**: Transparent or white background
- **File naming**: Use lowercase with the exact filenames listed above

## How to Save Logos

1. Download the official brand logos from:
   - Apple: https://www.apple.com/newsroom/media-library/
   - Samsung: https://news.samsung.com/global/brand-assets
   - OnePlus: https://www.oneplus.com/support/brand-assets
   - Vivo: https://www.vivo.com/en/about-vivo/brand-assets
   - Oppo: https://www.oppo.com/en/about/brand-assets

2. Or search for "brand name logo png" on Google Images and filter by:
   - Size: Large
   - Type: Transparent
   - Usage rights: Labeled for reuse

3. Save each logo with the exact filename listed above in this folder: `frontend/public/media/brand-logos/`

4. Ensure the logos are high quality and have transparent backgrounds for best appearance.

## Alternative: Using Online Logo Services

You can also use services like:
- LogoSearch: https://logosearch.io/
- Brandfetch: https://brandfetch.com/
- Clearbit Logo API: https://clearbit.com/docs#logo-api
